# Retail Sales Power BI Dashboard

## Project Overview
This project presents an interactive Power BI dashboard built using a cleaned retail sales dataset.
The objective is to communicate business insights effectively through visual storytelling.

## Tools Used
- Power BI Desktop
- CSV Dataset
- Data Visualization Techniques
- Exploratory Data Analysis (EDA)

## Repository Structure
- data/ : Dataset used for analysis
- reports/ : Power BI report file
- screenshots/ : Dashboard screenshots
- docs/ : Complete project documentation

## How to Use
1. Download the Power BI (.pbix) file from the reports folder
2. Open using Power BI Desktop
3. Explore visuals using slicers and filters
